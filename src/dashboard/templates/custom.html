Strategy Info

Custom bot uses a configurable strategy with Smart Flip:

  • Signals: Sweep + Displacement only (threshold 60) — no pullback gate
  • Flip mode: {{ custom_flip_mode|replace('_', ' ')|title }} — adjustable above
  • Smart Flip: Uses a 6-factor model (funding rate, L/S ratio, crowding, liquidation proximity, kill zone timing, OI) to compute per-signal sweep probability. Only flips when probability exceeds the threshold.
  • Margin: {{ (custom_margin_pct * 100)|round(0)|int }}% of balance per trade — adjustable via slider above
  • Independent scan: Runs its own {{ custom_scan_interval }}-minute cycle, separate from main and flipped bots
  • Progressive TP: 33% at 1R, 33% at 2R, remaining 34% trails via ATR. SL moves to breakeven after TP1.
  • Cooldown: Only applies after stop-loss exits (4h). TP/trailing stop exits allow immediate re-entry.
