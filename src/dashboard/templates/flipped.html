Strategy Info

Flipped bot uses a DIFFERENT, SIMPLER strategy than the main bot:

  • Signals: Sweep + Displacement only (threshold 60) — no pullback gate
  • No leverage intelligence: Does not use funding rate data
  • No quality whitelist: Scans ALL tradeable pairs
  • Direction flip: Bullish signals traded as SHORT, bearish as LONG
  • Independent scan: Runs its own 15-minute cycle, separate from main bot
  • Trailing exits: TP acts as milestone — SL moves to breakeven, then trails via ATR. No fixed TP exits.

Main bot: TTC strategy (sweep + displacement + pullback, threshold 70, quality whitelist, leverage intelligence)
